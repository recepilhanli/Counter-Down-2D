# Counter Down 2
![GitHub Release](https://img.shields.io/badge/Released-22.11.23-blue)


## Description
an unofficial 2D version of Counter Strike

## Controls

**W,A,S,D** -> Movement

**Mouse** -> Rotation

**B** -> Buy Armour

**M** -> Team Selection

**Left Mouse Button** -> Fire


## Features

Multiplayer

Bot System
